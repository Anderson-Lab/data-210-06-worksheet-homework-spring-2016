    Setup

    1. Go into virtual box and add a port forward from port 8000 to port 8000 using 127.0.0.1. This can be done by modifying the file called Vagrantfile in your MyDataScienceToolbox folder. Uncomment out the line (remove #) that forwards the port 8888 to 8888 on 127.0.0.1. Specifically, add this line: config.vm.network "forwarded_port", guest: 8000, host: 8000, ip: "127.0.0.1"
    2. Then type vagrant reload
    3. sudo apt-get install python3-setuptools
    4. sudo easy_install3 pip
    5. sudo npm install -g configurable-http-proxy
    6. sudo pip3 install ipython
    7. sudo apt-get install python3-dev
    8. sudo pip3 install notebook
    9. sudo pip3 install jupyterhub
    10. sudo pip3 install bash_kernel
    11. cd /usr/local/share/jupyter
    12. sudo wget http://freyja.cs.cofc.edu/kernels.tar.gz
    13. sudo tar xfvz kernels.tar.gz
    14. cd
    15. sudo apt-get install libzmq3-dev
    16. R
    17. Then type: install.packages(c('repr', 'IRkernel', 'IRdisplay'),repos = c('http://irkernel.github.io/', 'http://cran.us.r-project.org'))
    18. Then type: IRkernel::installspec()
    19. q()
    20. We are finally ready! If all has gone well, type jupyterhub, and then within Chrome go to http://localhost:8000
